# v0.1.0 - Initial Release

## Summary

Initial public release of the application with the core restaurant logging, mapping, authentication, and realtime feature set in place.

## Features

- Restaurant CRUD
- Map integration
- Geocoding
- Realtime updates
- Auth (Google + email)
- Theme support

## Technical Highlights

- Laravel 13 + PHP 8.3
- Vite build pipeline
- Redis queue and cache
- Reverb / Pusher protocol WebSocket

## Notes

- First stable baseline
- Production-ready configuration
